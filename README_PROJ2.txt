Algorithm 1 - algorithm1.py
--------------------------------------------------------------------------------------------------------------------------------
* This algorithm takes 2 arrays as input per test case: 
The first array is a single concatenation of some notable cities in California. 
The second array contains names of some real or imaginary cities, that may be present in the first array.
The output array contains starting indices of the target words in an array


* To run the algorithm, make sure you have Python installed on your computer. You can download Python at 
  https://www.python.org/downloads/.

To run Algorithm1.py, follow these steps:
* Open terminal
* Navigate to the directory containing the .py file(s) 
* Execute by typing python3 followed by the filename.
  Ex: python3 Algorithm1.py
* To modify the input, replace the text file with 2 arrays for each test case. Then, execute again.
